# 自动下载自选股数据的crontab配置
# 同时支持A股/港股和美股市场

# A股/港股市场（北京时间收盘后16:30执行）
30 16 * * 1-5 cd /Users/jijunchen/Documents/Projects/Chanlun_Bot && /usr/bin/python3 auto_download_watchlist_data.py today

# 美股市场（北京时间次日早上9:30执行，对应美股前一交易日收盘后）
30 9 * * 2-6 cd /Users/jijunchen/Documents/Projects/Chanlun_Bot && /usr/bin/python3 auto_download_watchlist_data.py today

# 注意：
# - 1-5 表示周一到周五（A股/港股交易日）
# - 2-6 表示周二到周六（对应美股周一到周五的交易日）
# - 这样配置可以确保所有市场的数据都能在交易日结束后及时下载
# - 请根据实际需求确认是否需要同时启用两行，或者只保留需要的市场
# - 确保Python路径正确（使用 which python3 查看实际路径）
# - 确保项目路径正确

# 安装方法：
# 1. 编辑crontab: crontab -e
# 2. 将上述配置添加到文件末尾
# 3. 保存并退出

# 验证crontab是否生效：
# crontab -l
