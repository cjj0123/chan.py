# 缠论交易助手 V2 - 用户指南

## 概述
缠论交易助手 V2 是一个全新的GUI界面，集成了数据更新、股票扫描、图表分析等功能，提供直观易用的操作体验。

## 功能特性

### 📈 扫描器选项卡
- **数据操作区**：更新本地数据库，支持自定义日期范围
- **扫描配置区**：选择时间级别（日线/30分钟/5分钟），支持仅扫描自选股
- **结果列表区**：显示扫描结果，包括代码、名称、信号类型、评分等
- **操作日志区**：实时显示操作日志

### 📊 图表分析选项卡
- **手动分析区**：输入股票代码进行单只股票分析
- **图表展示区**：显示缠论分析图表
- **分析详情区**：显示详细的分析结果

### ⚙️ 设置 & 自动化选项卡
- **基础设置**：笔严格模式开关，数据源配置
- **自动化模块**：港股自动交易、Futu实时监控（待实现）

## 使用方法

### 1. 启动应用
```bash
cd App
python TraderGUI.py
```

### 2. 更新数据库
1. 在"扫描器"选项卡中，设置日期范围
2. 选择时间级别（日线/30分钟/5分钟）
3. 点击"更新本地数据库"按钮
4. 观察进度条和日志输出

### 3. 执行扫描
1. 在"扫描器"选项卡中，选择时间级别
2. 可选择"仅扫描自选股"（可选）
3. 点击"开始扫描"按钮
4. 查看扫描结果和日志

### 4. 单只股票分析
1. 切换到"图表分析"选项卡
2. 输入股票代码（如：SH.600000）
3. 点击"加载图表"按钮
4. 查看生成的图表和分析详情

## 技术架构

### 线程安全
- 所有耗时操作都在后台线程中执行
- 使用Qt信号槽机制确保UI响应性
- 支持取消长时间运行的任务

### 数据源
- 主要使用SQLite数据库进行离线分析
- 支持多种时间级别（日线、30分钟、5分钟等）
- 兼容富途、Baostock等多种数据源

### 缠论分析
- 集成完整的缠论分析引擎
- 支持笔、线段、中枢的自动识别
- 提供买点/卖点检测功能

## 错误处理
- 完善的异常捕获和错误提示
- 详细的日志记录
- 优雅的任务取消机制

## 依赖项
- PyQt6
- matplotlib
- pandas
- 项目内部的缠论分析模块
