## 🐛 修复的问题

### 问题 1：期末持仓价值计算错误

**现象**：回测结果显示最终资金 17,743 HKD，看起来亏损 82%

**原因**：`final_funds` 只计算了现金余额，未包含未平仓持仓的价值

**修复**：
```python
# 修复前
results['final_funds'] = self.broker.available_funds

# 修复后
total_funds = self.broker.available_funds
for code, qty in self.broker.positions.items():
    if qty > 0:
        last_price = klines_30m[-1].close
        total_funds += qty * last_price
results['final_funds'] = total_funds
```

**验证**：
- 修复前：17,743 HKD（-82.26%）
- 修复后：100,753 HKD（+0.75%）✓
