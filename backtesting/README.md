# 回测系统说明

## 系统概述

本目录包含Chanlun Bot项目的回测系统，用于验证和优化交易策略。

## 回测系统对比

### 增强版回测 (`enhanced_backtester.py`) - 【推荐使用】

**特点**：
- **港股专业化**：精确计算港股交易成本
  - 佣金：0.03% (最低3港元)
  - 印花税：0.1% (向上取整，买卖双边)
  - 交易费：0.00565% (买卖双边)
  - 中央结算费：0.002% (买卖双边)
- **参数优化**：内置网格搜索功能
- **详细分析**：提供夏普比率等高级绩效指标

**适用场景**：
- 精确的港股策略回测
- 参数优化和调优
- 详细的风险收益分析
- 准备实盘前的深度验证

**使用方法**：
```bash
python3 enhanced_backtester.py --mode optimize
```

### 基础回测 (`backtester.py`)

**特点**：
- **基础功能**：提供基本的回测框架
- **通用性**：适用于多种市场
- **简单易用**：适合快速验证基本策略

**适用场景**：
- 初步策略验证
- 基础性能指标评估
- 快速迭代测试

## 推荐使用策略

**一般情况下推荐使用增强版回测**，因为它能够：
- 精确模拟港股交易成本
- 提供更真实的回测结果
- 支持参数优化以提升策略表现
- 生成详细的绩效分析报告

**特殊情况使用基础回测**：
- 需要快速验证策略概念时
- 测试非港股市场的策略时
- 不需要精确成本计算的初步测试时

## 注意事项

1. 由于Chanlun Bot项目主要交易港股，建议优先使用增强版回测
2. 增强版回测的结果更贴近实际交易表现
3. 参数优化功能可以帮助找到最佳策略参数组合
